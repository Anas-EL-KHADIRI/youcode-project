#  Sell4All Data Exploration Project

##  Project Overview

This project was completed as part of the **YouCode selection process**

The objective is to explore, analyze, visualize, and clean a customer dataset using **Python**, **Pandas**, and **Matplotlib** while following good software engineering practices with **Git** and **GitHub**

---

##  Project Objectives

The project consists of the following tasks:

- Read the CSV dataset using Pandas
- Display the first five rows of the dataset
- Generate and explain the dataset summary
- Calculate descriptive statistics (mean and median)
- Calculate the median age by country (bonus)
- Create a bar chart showing customer spendings by country
- Remove customers with spendings lower than **€10**
- Remove duplicate rows
- Export the cleaned dataset with the required columns

##  Dataset

The original dataset contains customer information such as:

- Name
- Phone Number
- Email
- Address
- Country
- Postal Code
- Last Date of Connection
- Last Time of Connection
- Age
- Gender
- Customer Spendings

After cleaning, only the following columns are kept:

- Country
- Age
- Gender
- Customer spendings

---

##  Data Analysis

The notebook includes:

- Dataset preview
- Technical summary using `df.info()`
- Mean calculation
- Median calculation
- Median age by country
- Customer spending analysis
- Bar chart visualization

---

##  Data Cleaning

The following cleaning operations were performed:

- Removed customers whose spendings are lower than **€10**
- Removed duplicate rows
- Selected only the required columns
- Exported the cleaned dataset into:

```
dataset-sell4all-cleaned.csv
```

##  Visualization

A bar chart was created to visualize the **total customer spendings by country**.

The visualization includes:

- Colored bars
- Spending values displayed above each bar
- Rotated country labels for better readability
